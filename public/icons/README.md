# App-Icons

Das Kompasszeichen in `inner-compass.svg` wurde als eigene geometrische Arbeit für Inner Compass erstellt. Es verwendet keine fremde Marke, Schrift, Icon-Bibliothek oder kopierte Vorlage.

Die PNG-Dateien sind Rasterableitungen derselben Geometrie:

- `inner-compass-192.png` und `inner-compass-512.png` für das Web-App-Manifest;
- `inner-compass-maskable-512.png` mit vollflächigem Hintergrund und Motiv innerhalb der Maskable-Safe-Zone;
- `favicon-32.png` als kleine Browservariante;
- `../apple-touch-icon.png` in 180 × 180 Pixeln für iOS.

Bei einer Änderung am Zeichen müssen alle Rastergrößen erneut erzeugt und visuell geprüft werden.
